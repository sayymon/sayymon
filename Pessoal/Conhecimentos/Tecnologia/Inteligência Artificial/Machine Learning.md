
**Machine Learning** é uma área da [[Inteligência Artificial]] que se baseia em [[Algoritmo]] e modelos estatísticos para permitir que sistemas computacionais aprendam com dados e façam previsões ou tomem decisões sem serem explicitamente programados. É uma das áreas mais promissoras da atualidade, com aplicações em diversas indústrias e campos.

## Principais Processos : 

- [[Coleta de Dados]]
- [[Feature Engineering]]
- [[Logging]]
- [[Gerenciamento dos Dados]]
- [[Automação ML]]
- [[Análise e Exploração dos Dados]]
- [[Treinamento do Modelo em Escala]]
- [[Monitoramento]]

## Principais Algoritmos

- [[Regressão Linear]]
- [[Regressão Logística]]
- Árvores de Decisão
- Random Forest
- Support Vector Machines (SVM)
- [[Redes Neurais]]
- K-Means Clustering
- Naive Bayes

## Tipos de [[Modelos de IA]]

Modelo de Aprendizado - Supervisionado (onde os dados históricos são rotulados)
- Regressão
- Classificação  
Não Supervisionado (Não tem histórico e precisa fazer uma analise exploratória)
- Clusterização
Reforço
- [[Deep Learning]]

A escolha do modelo correto é essencial para o sucesso do projeto

## Aplicações de Machine Learning

- Reconhecimento de Padrões
- Análise de Sentimentos
- Recomendação de Produtos
- Detecção de Fraudes
- Previsão de Vendas
- Diagnóstico Médico
- Veículos Autônomos

## Desafios em Machine Learning

- [[Overfitting]]
- [[Underfitting]]
- Viés e Variância
- Interpretabilidade dos Modelos
- Dados Desbalanceados
