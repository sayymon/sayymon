
O Spring é um framework famoso no eco sistema Java, ficou muito famoso na época por facilitar demais configurações e integrações devido os diversos módulos que eram aplicados a tudo.

Em meados de 2014 veio o [[Spring Boot]] que era revolucionário para o mundo java buscando trazer facilidades para o java que outras linguagens já estavam nascendo e evoluindo.

Junto do Spring boot veio o [[Spring Initializr]] que é sempre muito pratico e usual integrado a diversas [[IDE]]'s com o [[Spring Tools]] que permite navegar nos recursos do Spring e ajuda a conferir configurações

Tem também o [[Spring CLI]] para quem curte fazer as coisas via terminal ou automatizar o processo de criação de projetos 

## Módulos Core

- Spring Core: fornece recursos básicos e essenciais para o desenvolvimento de aplicações Spring.
- Spring Beans: oferece suporte para a criação e gerenciamento de beans na aplicação.
- Spring Context: fornece funcionalidades avançadas para o contexto de aplicação, como injeção de dependências e internacionalização.

## Módulos Web

- [[Spring MVC]]: framework web que facilita o desenvolvimento de aplicações web baseadas em padrões MVC.
- [[Spring WebFlux]]: oferece suporte para programação reativa no desenvolvimento de aplicações web.
- [[Spring Web Services]]: facilita a criação e consumo de serviços web baseados em padrões SOAP.
- [[Spring HATEOAS]] : Maneira de Expor endpoints utilizando os verbos http e navegabilidade entre os recursos

## Módulos Data Access

- [[Spring JDBC]]: fornece suporte para acesso a banco de dados utilizando JDBC puro.
- [[Spring ORM]]: integração com frameworks ORM como Hibernate, JPA, etc.
- [[Spring Data]]: simplifica o acesso a bancos de dados relacionais e não relacionais.
- [[Spring GraphQL]] : Viabiliza integração via GraphQL

## Módulos Integration

- [[Spring Integration]]: facilita a integração de sistemas heterogêneos através da definição de fluxos de integração.
- [[Spring Batch]]: facilita o desenvolvimento e execução de processos batch na aplicação.
- [[Spring AMQP]]: oferece suporte para integração com sistemas que utilizam mensageria baseada em AMQP.

## Módulos Security

- [[Spring Security]]: framework que oferece recursos avançados para implementar autenticação, autorização e proteção contra ataques na aplicação.
- [[Spring Authorization Server]] : Prove implementações de OAuth 2.0 e OpenId Connect
