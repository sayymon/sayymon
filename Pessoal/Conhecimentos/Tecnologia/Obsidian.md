### O que é o Zettelkasten ?

O [[Zettelkasten]] é um sistema de organização de notas desenvolvido pelo sociólogo alemão Niklas Luhmann. Consiste em criar e interligar pequenos blocos de notas, chamados zettels, de forma a facilitar o processo de escrita e pesquisa. 

Cada zettel contém uma única ideia ou informação, e é indexado para ser facilmente encontrado e conectado a outros zettels relacionados. 

Essa abordagem ajuda a estimular a criatividade, promover insights inesperados e organizar o conhecimento de forma mais eficiente.

# Atalhos e dicas de Uso : 

Tecla `command + N` = Nova Nota
Tecla `command + O` = Abrir uma nota Existente
Tecla `command + F` = buscar uma informação na Nota atual
Tecla `command + j` =  Completar usando o Chat GPT plugin Text Generator
Tecla `command + t` = Abrir Nova Aba
Tecla `command + w` = Fechar Aba Atual
Tecla `command + shift + i` = Inserir uma Imagem

Para criar tags basta colocar o #tag

# Tags

## Para que servem as tags no obsidiam ?


As tags no Obsidian servem para categorizar e organizar as notas de forma mais eficiente. Ao adicionar tags a uma nota, é possível agrupar e filtrar notas relacionadas com base nessas tags, facilitando a busca por informações específicas e a criação de conexões entre os diferentes zettels. Além disso, as tags também podem ajudar na organização do seu sistema de notas, permitindo uma visão mais clara e estruturada do conhecimento armazenado no Zettelkasten.

## Quando usar tags ao invés de links internos ?


As tags podem ser úteis quando se deseja categorizar e agrupar notas que abordam um mesmo tema ou conceito, facilitando a organização e a busca por informações específicas. Por outro lado, os links internos são mais adequados para estabelecer conexões diretas entre notas relacionadas, permitindo uma navegação mais fluida e aprofundada no sistema de notas. Portanto, o uso de tags ou links internos depende do objetivo específico de organização e conexão de informações dentro do Zettelkasten.
