
A AWS tem esse serviço que auxilia a planejar estratégias de Backup junto as politicas de recuperação de Desastres

O AWS Backup é um serviço de proteção de dados totalmente gerenciado que facilita a centralização e a automatização em serviços da AWS na nuvem e locais. 

Você pode usar o AWS Backup para automatizar e consolidar as tarefas de backup que foram executadas anteriormente um serviço por vez, sem a necessidade de criar scripts personalizados e processos manuais.
