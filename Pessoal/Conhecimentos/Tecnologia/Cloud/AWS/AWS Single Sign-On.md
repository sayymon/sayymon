
[[Single Sign-On (SSO)]], ou Autenticação Única, é um sistema de gerenciamento de identidade que permite aos usuários acessar múltiplos aplicativos e serviços com um único conjunto de credenciais de login.  Em vez de ter que lembrar diferentes nomes de usuário e senhas para cada aplicativo, os usuários fazem login uma vez e ganham acesso a todos os recursos autorizados.

No contexto da AWS, o AWS SSO é um serviço que permite centralizar o gerenciamento de acesso a várias contas da AWS e aplicativos de negócios.  Ele simplifica o processo de login para os usuários e fornece aos administradores um controle granular sobre as permissões de acesso.


**Em resumo:**  Imagine uma chave mestra que abre várias portas. O SSO é essa "chave mestra" para seus aplicativos e serviços, permitindo acesso com um único login.  Isso melhora a segurança, facilita a gestão de usuários e aumenta a produtividade, eliminando a necessidade de múltiplos logins.
