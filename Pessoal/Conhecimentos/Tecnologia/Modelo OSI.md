Title: Modelo OSI

O Modelo OSI (Open Systems Interconnection) é um modelo conceitual que descreve como diferentes sistemas se comunicam em uma rede. Ele divide a comunicação em sete camadas, cada uma com responsabilidades específicas, facilitando a padronização e a resolução de problemas.  Imagine como uma linha de montagem para enviar uma mensagem: cada camada adiciona sua parte até chegar ao destino, onde o processo é revertido.

As 7 camadas do Modelo OSI são:

1. **Camada Física (Physical Layer):**  Transmissão de bits brutos pelo meio físico (cabos, fibra óptica, etc.).  Lida com características elétricas e físicas da conexão. *Exemplos: cabos, hubs, repetidores.*

2. **Camada de Enlace de Dados (Data Link Layer):**  Fornece comunicação confiável entre dois nós diretamente conectados. Divide os dados em frames e controla o fluxo de dados.  *Exemplos: switches, pontes.*

3. **Camada de Rede (Network Layer):**  Encaminhamento de pacotes entre redes diferentes.  Determina o melhor caminho para os dados. *Exemplos: roteadores, protocolos IP.*

4. **Camada de Transporte (Transport Layer):**  Garante a entrega confiável e ordenada dos dados entre as aplicações. Controla o fluxo e a correção de erros. *Exemplos: protocolos TCP e UDP.*

5. **Camada de Sessão (Session Layer):**  Gerencia a comunicação entre aplicações, estabelecendo, gerenciando e encerrando sessões. *Exemplos: controle de login e logout.*

6. **Camada de Apresentação (Presentation Layer):**  Formata os dados para que as aplicações possam entendê-los.  Lida com criptografia, compressão e conversão de formatos. *Exemplos: SSL, TLS.*

7. **Camada de Aplicação (Application Layer):**  Interface direta com o usuário e suas aplicações.  Fornece serviços de rede para os programas. *Exemplos: HTTP, FTP, SMTP, DNS.*
