
O Java é a [[Linguagem de Programação]] que trabalho desde 2010 é uma linguagem de propósito geral amplamente utilizada para o sistemas embarcados em uma variedade de cenários.

Normalmente utiliza o paradigma de [[Orientação a Objetos]], mas também suporta uma [[Programação Estruturada]] e mais recentemente a partir das versões 8 com (lambda functions e streams) vem cada vez mais se tornando uma linguagem mais funcional suportando inclusive [[Programação Funcional]]

Para começar a programar em Java, devemos instalar e configurar o JDK (Java Development Kit) que pode ser instalado utilizando um gerenciador como o [[Sdkman]] o [[asdf]] ou instalando a versão mais recente também através do [Site da Oracle](https://www.oracle.com/java/technologies/downloads/) . 

Normalmente utilizamos uma  [[IDE]] como [[IntelliJ IDEA]], [[Vscode]] e algum tempo atras o [[Eclipse]] e o [[NetBeans]] para dar mais produtividade.

## Evolução e features por versão do Java

- Java 5: introdução de generics, foreach loop, autoboxing/unboxing
- Java 6: melhorias no desempenho, suporte para web services
- Java 7: introdução de strings em switch, try-with-resources, diamond operator
- Java 8: introdução de lambda expressions, streams, métodos default em interfaces
- Java 9: modularização do código com Jigsaw, melhorias na API de processamento de streams
- Java 10: introdução da inferência de tipo local (var)
- Java 11: LTS (Long-Term Support) release, remoção do Java EE e Corba modules
- Java 12: introdução do switch expressions
- Java 13: melhorias na performance e estabilidade
- Java 14: introdução das expressões de record para facilitar a criação de classes imutáveis
- Java 15: melhorias na performance e estabilidade, preview features como sealed classes e pattern matching for instanceof
- Java 17: LTS (Long-Term Support) release, introdução do tipo de dados record, padrões de correspondência em switch aprimorados, melhoria na API de processamento de strings
- Java 21: Melhorias na performance e estabilidade, preview features como sealed classes e pattern matching for instanceof
- Java 22: Introdução do novo sistema de tipo primitivo int8 para otimizar o uso da memória, melhorias na API de coleções
