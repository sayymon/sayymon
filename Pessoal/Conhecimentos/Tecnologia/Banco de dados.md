
## Banco de Dados

Um banco de dados é uma coleção organizada de informações, ou dados, tipicamente armazenados eletronicamente em um sistema de computador. Serve para armazenar, organizar, recuperar e manipular dados de forma eficiente.  Utilize-o quando precisar guardar e acessar informações de forma estruturada, confiável e escalável.  Conceitos principais: tabelas, registros, campos, chaves primárias, chaves estrangeiras, índices,  consultas (SQL), transações e segurança.


**Tipos de Bancos de Dados:**

| Tipo                               | Descrição                                                                 | Uso                                                                                                         |
| ---------------------------------- | ------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **Relacionais (SQL)**              | Dados organizados em tabelas com relações entre elas.                     | Aplicações tradicionais de negócios, sistemas de gerenciamento, e-commerce.  Ex: MySQL, PostgreSQL, Oracle. |
| **Chave-Valor (NoSQL)**            | Dados armazenados como pares chave-valor. Simples e rápido.               | Cache, sessões de usuário, jogos online. Ex: Redis, Memcached.                                              |
| **Documento (NoSQL)**              | Dados armazenados em documentos (JSON, XML). Flexível e sem esquema fixo. | Aplicações web modernas,  CMS,  e-commerce. Ex: MongoDB, Couchbase.                                         |
| **Grafo (NoSQL)**                  | Dados armazenados como nós e relacionamentos (arestas).                   | Redes sociais, recomendações, análise de links. Ex: Neo4j, Amazon Neptune.                                  |
| **Em memória (In-Memory)**         | Dados armazenados na memória RAM para acesso ultra-rápido.                | Cache, processamento de transações em tempo real, análise de dados em alta velocidade. Ex: Redis, SAP HANA. |
| **Séries Temporais (Time Series)** | Dados indexados por tempo, otimizados para análise de séries temporais.   | Monitoramento de sistemas, IoT, dados financeiros. Ex: InfluxDB, TimescaleDB.                               |
| **Ledger (Blockchain)**            | Dados armazenados em um registro imutável e distribuído.                  | Criptomoedas, contratos inteligentes,  registros auditáveis. Ex: Hyperledger Fabric, Corda.                 |
