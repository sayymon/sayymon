
K9s é um programa de interface de linha de comando ([[CLI]]) para interagir com clusters [[Kubernetes]]. Ele fornece uma maneira mais eficiente e intuitiva de visualizar e gerenciar recursos de Kubernetes, como [[pods]], [[deployments]] e serviços. Com o K9s, os usuários podem facilmente navegar e executar operações em seus [[clusters]] Kubernetes diretamente da linha de comando, sem a necessidade de usar várias ferramentas ou interfaces gráficas. 

É uma ferramenta poderosa e flexível para desenvolvedores e administradores que trabalham com Kubernetes.

## Como instalar no mac ? 



Para instalar o K9s no seu Mac, você pode utilizar o Homebrew, um gerenciador de pacotes para macOS. Basta abrir o terminal e executar o seguinte comando:

```
brew install derailed/k9s/k9s
```

Após a instalação, você pode começar a usar o K9s digitando `k9s` no terminal. Certifique-se de que seu cluster Kubernetes esteja configurado corretamente para que o K9s possa se conectar a ele.
