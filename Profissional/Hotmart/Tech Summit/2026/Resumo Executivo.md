# Resumo Executivo - Tech Summit 2026

## Informações do Evento
- **Data**: 02 a 06 de Fevereiro de 2026
- **Local**: Belo Horizonte, MG
- **Hotel**: Mercure Lourdes (Av do Contorno)
- **Participação**: Presencial

## Principais Temas

### 1. Inteligência Artificial
O tema dominante do evento. Praticamente todas as áreas apresentaram iniciativas com IA:
- Nova arquitetura de agentes (AWS Agent Core, LiteLLM, LangFuse)
- Múltiplos agentes: SARA, Bifrost, CAIO, Sales Assistant, CS Assistant
- Hotmart AI Companion (Concierge)
- Cosmos AI + Figma MCP

### 2. Encontrabilidade de Dados
Programa estratégico de Data Platform para 2026:
- 8.5k tabelas Silver, 845 Gold, 2k Diamante
- Foco em metadados: Domínio, Owners, Descrição, Tags
- Ferramentas: DataHub, Astrolens, Astrobox, DataBot
- Heimdal de Data para métricas

### 3. Desacoplamento e Modernização
- MarketplaceDB Decoupling (Strangler Fig Pattern)
- SAP B1 → SAP S/4HANA (economia 1,30M BRL)
- Drone → Github Actions (deadline: 31/03/2026)
- MongoDB → DocumentDB (economia potencial 20K USD/mês)
- app.vulcano deprecation

### 4. Platform Reliability
- LCP: 22s → 16s → 6.9s
- 40+ microfrontends
- Monorepo approach
- Adoção massiva do Cosmos
- SSR implementation

### 5. DevOps e GitOps
- ArgoCD como ferramenta principal
- Canary Deployment
- TechDeck expansion
- Multiregion
- Karpenter (economia 92K USD/ano)

### 6. Cyber Security
- Blue Team, Red Team, NSOC
- 6 focos: Metrics, Posture, Awareness, Enabler, Risk, Compliance
- Programa técnico: AppSec, Awareness, Diminuir Risco
- Atenção a CVEs e GDVs

### 7. Custos e Otimização
- Economia 2025: 11.4M BRL (4.1M Saving + 7.3M Cost Avoidance)
- 73% de infra não utilizada ($83K/mês oportunidade)
- Otimização de modelos IA (GPT5-nano, Gemini)
- Karpenter em 60 clusters
