# Cosmos

## Visão Geral
Design System da Hotmart que facilita e otimiza a criação de interfaces consistentes e padronizadas.

## Contexto
- **Apresentado por**: [[Lucas Miranda]]
- **Evento**: [[Tech Summit 2026]]
- **Programa**: [[Cosmos Adoption]]

## Componentes

### Cosmos Core
Base do design system com componentes fundamentais.

### Cosmos Global
Evolução do Cosmos Core com componentes reutilizáveis e modulares.
- Frontend modular
- Componentes reutilizáveis
- Experiências personalizadas

## Inovações 2026

### Figma MCP
- Integração com Figma via Model Context Protocol
- Importação automática de designs
- Conversão de Figma para código

### Cosmos AI
Funcionalidade revolucionária que permite:
- Uso em IDEs (Kiro, Cursor, Manus)
- Configuração via MCP do Cosmos
- Importação do modelo do Figma com formato Cosmos
- Criação automática de frontends seguindo padrões e golden paths da Hotmart
- Código praticamente funcional gerado automaticamente

## Objetivo Estratégico
Democratizar a criação de frontends, permitindo que mais pessoas criem interfaces seguindo os padrões da Hotmart sem necessidade de conhecimento profundo de frontend.
