# AstroboxAutomations

## Visão Geral

Sistema de automação que recebe e processa leads de diferentes fontes, aplicando regras de qualificação e roteamento.

## Responsabilidades

- Recebimento de leads via webhooks
- Aplicação de regras de qualificação
- Classificação de origem (Paid, Organic, Cold)
- Roteamento para sistemas downstream
- Persistência de dados
